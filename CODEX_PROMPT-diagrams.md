diagrams, and **Prompt B** to fix the PowerPoint diagrams. (Codex also reads `AGENTS.md` for house rules.)

> ### ⚠️ What is SOURCE vs. GENERATED — edit source only
> - **HTML files** (`index.html`, `teaching-guide-app.html`, `gut-brain-axis-indepth.html`, …) **are
>   source** — edit them directly. (Prompt A.)
> - **`build-deck.js` is the source for the slide deck** — edit *this* to change the deck. (Prompt B.)
> - **Do NOT edit `Gut-Brain-Axis-Deck-Depth.pptx` or the `slides/*.jpg` images by hand** — they are
>   *generated outputs*. The `.pptx` is rebuilt by `node build-deck.js`; the `slides/` images are
>   re-rendered from the `.pptx`. Hand-editing them would just be overwritten on the next build.
>
> So: **for the web pages, edit the `.html`; for the deck, edit `build-deck.js` then regenerate + re-render.**
